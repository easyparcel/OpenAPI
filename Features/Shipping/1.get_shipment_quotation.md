### Get Shipment Quotation

Get Shipment Quotation: This feature enables users to obtain shipment quotations from all courier companies available on the EasyParcel platform. Users need to provide detailed sender and receiver addresses to get accurate quotations.

#### [Feature/Endpoints](../README.md)  |  [Back to official documentation](../../README.md)

#### Endpoint URL:

```
https://api.easyparcel.com/shipment/quotations
```

### [Request](#request-parameters) | [Response](#response-parameters)

---

### Request Parameters

| Field                     | Type        | Required | Description                            | Remarks                                                       |
| ------------------------- | ----------- | -------- | -------------------------------------- | ------------------------------------------------------------- |
| sender.postcode           | string(10)  | Yes      | Sender's postcode                      |                                                               |
| sender.subdivison\_code   | string(35)  | Optional | Sender's state ISO 3166-2 code         | Refer to [ISO\_3166.md](../../References/ISO_3166.md)         |
| sender.country            | string(2)   | Yes      | Sender's country code                  | Refer to [country\_code.md](../../References/country_code.md) |
| receiver.postcode         | string(10)  | Yes      | Receiver's postcode                    |                                                               |
| receiver.subdivison\_code | string(35)  | Optional | Receiver's state ISO 3166-2 code       |                                                               |
| receiver.country          | string(2)   | Yes      | Receiver's country code                |                                                               |
| parcel\_value             | double(8,2) | Optional | Declared parcel value for DDP/COD/etc. |                                                               |
| weight                    | double(8,2) | Yes      | Weight in KG                           |                                                               |
| width                     | double(8,2) | Optional | Width in CM                            |                                                               |
| height                    | double(8,2) | Optional | Height in CM                           |                                                               |
| length                    | double(8,2) | Optional | Length in CM                           |                                                               |

#### Sample Request

```json
{
  "list": [
    {
      "sender": {
        "postcode": "10150",
        "subdivison_code": "MY-02",
        "country": "MY"
      },
      "receiver": {
        "postcode": "14000",
        "subdivison_code": "MY-02",
        "country": "MY"
      },
      "parcel_value": 50,
      "weight": 0.5,
      "width": 5,
      "length": 5,
      "height": 5
    }
  ]
}
```

---

### Response Parameters

| Field        | Type   | Description               |
| ------------ | ------ | ------------------------- |
| status\_code | int    | HTTP-like status code     |
| message      | string | Status description        |
| data         | array  | List of quotation results |

Each item in `data[]` includes:

#### Input

Same format as request input.

#### Quotations

Each courier quotation contains:

| Field                      | Type        | Description                               |
| -------------------------- | ----------- | ----------------------------------------- |
| courier.service\_id        | string      | Unique service ID                         |
| courier.service\_name      | string      | Courier service name                      |
| courier.courier\_id        | string      | Courier unique ID                         |
| courier.courier\_name      | string      | Courier company name                      |
| courier.courier\_logo      | string      | Logo URL                                  |
| courier.delivery\_duration | string/null | Delivery duration                         |
| courier.service\_tag       | array       | Array of service label/destination/method |

#### Pricing

| Field                  | Type   | Description          |
| ---------------------- | ------ | -------------------- |
| currency               | string | Price currency       |
| total\_amount          | string | Final total          |
| shipment\_price        | string | Base shipping price  |
| total\_features\_price | string | Add-on features cost |

#### Features (Dynamic based on courier)

Includes:

* `cod`
* `sms_tracking`
* `email_tracking`
* `whatsapp_tracking`
* `awb_branding`
* `ddp_charges`

Each feature object includes relevant subfields (e.g., `sms_price`, `banner_price`, `import_tax_charges`, etc.).
